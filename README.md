# data-analysis-browser

A data analysis web browser with a text box to enter or load data as a csv file and different menu options for different  data analysis operations to be carried out on the data after pressing a button. The data will then be displayed.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/data-analysis-browser.git
cd data-analysis-browser
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
